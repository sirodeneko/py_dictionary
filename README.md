# py_dictionary
一个用py写的简单的英文字典程序，界面清新，操作简单，功能精巧。

## 使用平台：
* Windowns
* Linux
* Mac OS

支持本地搜索

## 文件说明
|文件						|说明					        |
|-----------------------|-----------------------------------|
| /db/EDB.db			|数据库文件             		    |
|/Sdet_core.py			|**词典核心功能文件	**				|
|/Control.py		    |**词典核心功能文件	**		        |
|/DB.py			        |操作数据库的文件			        |
|/GUI.py			    |**GUI界面显示文件**			   |




## 使用方法
1. 安装python3.x
2. 运行GUI.py
